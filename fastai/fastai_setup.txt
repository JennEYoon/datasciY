tilda "~/" seems to be same as dot-dot "..\" in windows.
Relative address, add up-path to current location.

"~$" by itself if the home directory.
"~/fastai$" is the first level subdir to home.

"!/usr/bin/env python"
would be a jupyter shell command to user's env directory, followed by script call to python execution file.
