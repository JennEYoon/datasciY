Based on Kaggle data on Titanic passengers. What determines the survival rate? Passengers' sex, age, wealth, and location of birth are studied. First data is downloaded from Kaggle website and massaged. Then exploratory data analysis is conducted on a sample data set. Based on the results, features are selected and compared against survival rate. Then logistic regression classifier is used in order to see if additional features can improve a survival prediction model. Resulting data and findings are presented in detailed graphs.
