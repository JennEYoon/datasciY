Titanic Project (ML)

Passenger information from the Titanic ship is a common data set used in machine learning (ML). Here I use Python and data science libraries to find patterns in the data and build a prediction model. Then I use various visualization libraries to create pretty figures.

  • Link to Project: Titanic
  • Steps, Tags: exploratory data analysis (EDA), machine learning (ML), graphics, logistic regression
  • Date Source: https://www.kaggle.com/c/titanic/data, go to Kaggle data.
  • Reference Source: Rossant, Cyrille, Ipython Interactive Computing and Visualization Cookbook, 2nd ed., Packt Publishing 2018, pp. 299-304.
